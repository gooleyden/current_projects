# Создание автоматизированного решения по обработке данных о привлеченных лидах компании и построению отчета о конверсии и эффективности маркетинговых каналов привлечения клиентов

## Задача: 
Разработать автоматизированное решение, которое:
1. Объединяет данные из разных листов файла Excel в одну таблицу;
2. Рассчитывает выручку в рублях и долларах;
3. Формирует отчет с отображением конверсии и анализа ключевых параметров (суммы выручек по регионам, причины дисквалификации лидов, эффективность привлечения клиентов по разным каналам).

## Результат:

Решение успешно обрабатывает данные и формирует отчет в виде pdf-файла.
