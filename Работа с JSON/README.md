# Извлечение массива данных из JSON

## Задание:
Выгрузить массив с характеристиками:
1. год издания
2. название работы (для каждой работы может быть несколько описаний)
3. значения идентификатора типа doi (значение externalIdentifierId при условии, что externalIdentifierType = doi)
4. значения идентификатора типа eid (значение externalIdentifierId при условии, что externalIdentifierType = eid)

## Ход работы:
1. Ознакомление со структурой данных (вывел все ключи (скриншот ниже), посмотрел, что в них содержится, дополнительно воспользовался сервисом https://jsonformatter.curiousconcept.com)

2. Написание функции для извлечения нужных данных: 
Создание списка, где элементы - работы, для каждой из которых:
  - находится год публикации
  - находится название работы (берется самое длинное, если для работы есть несколько описаний)
  - записываются значения идентификаторов doi и eid отдельно (если среди идентификаторов нет нужного, то значение этого параметра остается незаполненным)

3. Проверка на корректность выгрузки:
  - количество работ в выгрузке и изначальном файле равны
  - нет пропусков в годе издания и названиях работ
  - годы издания соответствуют реальным
  - все идентификаторы уникальны и не повторяются
  - идентификаторы начинаются одинаково (10. для doi, 2 для eid)

4. Вывод на экран количества выгруженных работ

6. Вывод на экран количества и индексов работ, у которых отсутствуют значения для обоих идентификаторов: doi и eid
  
8. Сохранение полученного массива в формате Excel
